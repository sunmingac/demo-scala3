# Demo Scala 3 Project

This is a template project using Scala 3 with the following features:
- sbt as the build tool
- munit for testing

## Prerequisites

- JDK 11 or higher
- sbt 1.x

## Getting Started

To run the project:

```bash
sbt run
```

To test the project:

```bash
sbt test
```
